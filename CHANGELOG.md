# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-29

### Added
- Initial release
- Waitlist management with 3 modes (prelaunch, gated, viral)
- Viral referral tracking with unique codes
- Tier-based reward system with position bumping
- Full analytics (K-factor, cohorts, A/B testing, channels)
- Webhook events with HMAC-SHA256 signing and retry
- Admin dashboard (React SPA)
- Headless SDK (`@waitlist/sdk`)
- Drop-in widget (`@waitlist/widget`)
- React components (`@waitlist/react`)
- Comprehensive test suite (466 tests)
